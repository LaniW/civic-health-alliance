  2. Inside the App() function body, after the `zoomTo` useCallback (after line ~742), add two refs and two callbacks:

     ```jsx
     const touchStartX = useRef(null);
     const touchStartY = useRef(null);

     const handleTouchStart = useCallback((e) => {
       touchStartX.current = e.touches[0].clientX;
       touchStartY.current = e.touches[0].clientY;
     }, []);

     const handleTouchEnd = useCallback((e) => {
       if (phase !== "idle") return;
       if (touchStartX.current === null) return;
       const deltaX = e.changedTouches[0].clientX - touchStartX.current;
       const deltaY = e.changedTouches[0].clientY - touchStartY.current;
       touchStartX.current = null;
       touchStartY.current = null;
       if (Math.abs(deltaX) <= Math.abs(deltaY)) return;
       const THRESHOLD = 50;
       if (deltaX > THRESHOLD && level > 0) zoomTo(level - 1, null);
       else if (deltaX < -THRESHOLD && level >= 0 && level < 2) zoomTo(level + 1, null);
     }, [level, phase, zoomTo]);
     ```

     Critical details:
     - `useRef(null)` stores mutable touch coordinates without triggering re-renders
     - `phase !== "idle"` guard prevents swipe during fade-out/fade-in animation (Pitfall 4 from RESEARCH.md)
     - `Math.abs(deltaX) <= Math.abs(deltaY)` prevents vertical LeftPanel scrolling from triggering navigation (Pitfall 3 from RESEARCH.md)
     - `THRESHOLD = 50` (pixels) prevents accidental taps from being detected as swipes
     - Swipe right (`deltaX > THRESHOLD`): back one level, but NOT below level 0 (title screen is level -1, accessed only via Home button)
     - Swipe left (`deltaX < -THRESHOLD`): forward one level, but ONLY from levels 0 and 1. Level 2 to 3 requires explicit person tap (cannot swipe into a story without selecting a person). Level -1 (title) requires BEGIN button.
     - `[level, phase, zoomTo]` dependency array: `level` and `phase` are state variables that change; `zoomTo` is memoized with `[]` so it is stable. This prevents stale closure issues (Pitfall 1 from RESEARCH.md).

  3. Wire the handlers onto the App root `<div>` (the one with `touchAction:"manipulation"`). Change from:
     ```jsx
     <div style={{width:"100vw",height:"100vh",overflow:"hidden",background:CREAM,position:"relative",touchAction:"manipulation"}}>
     ```
     to:
     ```jsx
     <div style={{width:"100vw",height:"100vh",overflow:"hidden",background:CREAM,position:"relative",touchAction:"manipulation"}} onTouchStart={handleTouchStart} onTouchEnd={handleTouchEnd}>
     ```
  </action>
