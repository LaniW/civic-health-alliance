In the `HospitalScene` component (starts around line 525), modify the person rendering loop (the `HOSPITAL_PEOPLE.map(p=>(...))` block, around line 563) to add:

**A. Invisible touch target rect inside each person `<g>`:**

Add a `<rect>` element as the FIRST child inside each person's `<g>` element, before `PersonFig` and `Pulse`. This rect must use `fill="transparent"` (NOT `fill="none"` — `fill="none"` does not receive pointer events in SVG).

The rect for each person:
```jsx
<rect x={p.x-6} y={p.y-2} width={12} height={12} fill="transparent" style={{cursor:"pointer"}}/>
```

This creates a 12x12 SVG unit hit area centered horizontally on the figure (p.x-6 to p.x+6) and covering from just above the head (p.y-2) to below the torso (p.y+10). At the minimum viewport (768px, 4.08 px/SVG unit), 12 * 4.08 = 48.96px, exceeding the 44px touch target minimum.

**B. Add onTouchEnd and onTouchCancel handlers on the person `<g>`:**

The current person `<g>` has `onMouseEnter`, `onMouseLeave`, and `onClick`. Add two new handlers to fix sticky hover state on touch screens:

- `onTouchEnd={()=>onHoverPerson(null)}`
- `onTouchCancel={()=>onHoverPerson(null)}`
